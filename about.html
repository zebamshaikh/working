About this Visualization

Hybrid structure

The project uses a Drill down strucure using parameters like jobs, average pay, mortality rate, population and rental costs acrooss the different states in USA, compare them with different states and with national average. The visualization is User lead since the user can select which factor he wants to visualize on and the choloropeth map will help in determining which states rank best in the factors chosen.

Scenes

Apart from the overall map view and horizontal bar chart, users can select factors in text boxes below to determine 3 best states as per their interest.Scenes are consistent using uniform colors and layout.

Annotations

Mouse scrollover annotation are provided for each state on the map with a semi-transparent rectangle stating the factors of the State. The Bar chart has a National Average annotation too.

Parameters

A dropdown menu gives the user the Factor on which to render the Choloropeth map and the Horizontal Bar chart. In the next scene below the User can use Factors as rank for a personal State Recommendation which renders Line charts for the top three states in that category.

Triggers

When the user selects a factor from the dropdown menu, the map changes effectively to highlight states which are leading in that factor with a cholorepeth map, similarly the bar chart also changes.
